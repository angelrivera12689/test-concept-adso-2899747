Sobre el Proyecto

Football Tournament Management System es una plataforma integral para la gestión de torneos de fútbol, diseñada para cubrir necesidades tanto de ligas amateur como profesionales. Permite administrar equipos, jugadores, partidos, árbitros y estadísticas, todo de manera eficiente y segura.

  • Escalable: Fácil de ampliar con nuevas funcionalidades.
  • Modular: Separación clara por módulos.
  • Auditoría: Campos para rastrear cambios y acciones.
  • Consistencia: Estructura estandarizada y trazable.

Módulos y Entidades

1. Torneos

  • tournament: id, code, name, start_date, end_date, auditoría
  • stadium: id, name, location, capacity, auditoría

2. Equipos y Jugadores

  • team: id, name, tournament_id, coach_id, auditoría
  • player: id, name, jersey_number, position, team_id, auditoría
  • coach: id, name, auditoría
  • team_staff: id, name, role, team_id, auditoría

3. Partidos y Resultados

  • match: id, tournament_id, home_team_id, away_team_id, stadium_id, referee_id, date, auditoría
  • goal: id, match_id, player_id, minute, goal_type, auditoría
  • penalty: id, match_id, player_id, penalty_type, minute, reason, auditoría
  • referee: id, name, auditoría
